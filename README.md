# Training Planner

Автор:Глеб бондаренко

## Описание

Это GUI-приложение для планирования тренировок. Оно позволяет добавлять тренировки с указанием даты, типа и длительности, фильтровать их по типу и дате, сохранять и загружать данные в формате JSON.

## Использование

1. Введите дату в формате ДД.ММ.ГГГГ.
2. Выберите тип тренировки из выпадающего списка.
3. Введите длительность в минутах.
4. Нажмите «Добавить тренировку» для добавления записи.
5. Для фильтрации выберите тип и/или дату и нажмите «Применить фильтр».
6. Можно сохранить данные в JSON или загрузить их из файла.

## Тестовые сценарии

- Добавление нескольких тренировок.
- Фильтрация по типу и дате.
- Сохранение и загрузка данных.
